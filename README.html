README.html

"Welcome to MyPainter"

A title object, whose id is t1.

"Canvas"

A canvas object, where users can draw on it.

"Brush Shape"

Different brush shapes.
There are three choices for brush shape,
Rectange,
Round,
Normal.
When painting with brush, the cursor will be "cell".

It would be more obvious if you draw a "sharp" angle.

"Color Selector"

There are six choices for brush shape,
Red,
Yellow,
Green,
Blue,
Purple,
Black

"Text Input"

The following two rows are "Font Menu".
The upper row for users to choose the size of text.
The lower row for users to choose the typeface of text.
The last row, the input box for input text,
and click Submit if you finish typing.
There will be an alert to guide you that click on the canvas for the location of input text.

"Eraser"

An ordinary eraser, after you click the eraser, your cursor will change from "cell" into "pointer".

"Reset"

An ordinary refreshing button.

"Undo/redo"

Some ordinary Un/Re-do buttons.

"Download"

An ordinary download button for save the image of the canvas.

"Note"

Cursor icon:
when painting : "cell"
when erasing : "pointer"
