"MyPainter"
a title object, whose id is t1.
"Canvas"
a canvas object, where users can draw on it.
"Ctx"
an object which users use it to draw on the canvas.
"addEventListener"
to add all the event of drawing.

1. canvas.addEventListener('mousemove',draw);
This function always implement when mouse moves
2. canvas.addEventListener('mousemove',ChangeCursor);
This function will change the cursor if we click eraser, the cursor will change to cell to pointer.
else the cursor will be an cell when drawing a line with colors, red, green, black
3. canvas.addEventListener('mouseup',()=>to add the current canvas into stack for redo and undo purposes

"Brush shape"

we use the linecapmand linejoin to implement this function.
there are three choices for brush shape,
rectange,
round,
triangle

It would be more obvious if you draw a "sharp" angle.


"Brush Color"

"Brush Colors"
there are three choices for brush shape,
red,
green,
black


"Other Function"

"Eraser"
if you click the eraser, users can use this eraser with the cursor of pointer shape.
"Reset"
reset all the painter.
"Uno/Redo"
to undo and redo the painter.
"Download"
to download the canvas as an image
